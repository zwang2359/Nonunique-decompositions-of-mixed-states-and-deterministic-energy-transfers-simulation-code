# Nonunique decompositions of mixed states and deterministic energy transfers

This repository contains the Python implementation and simulation code for the paper: **"Nonunique decompositions of mixed states and deterministic energy transfers"**.

## 📌 Project Overview
This project simulates the models we used as examples in the paper, such as Jaynes-Cummings model, Buck-Sukumar model, Susskind–Glogower operator and Wigner phase space distribution. It uses Jupyter Notebooks to handle data processing, mathematical modeling, and visualization.

## 📁 Repository Structure
* `BS Model.ipynb` - The primary Jupyter Notebook containing the simulation code, equations, and data visualizations. Other files are similar.
* `requirements.txt` - The complete list of Python packages and specific versions required to replicate this environment.

## 🚀 How to Run the Simulation

### 1. Clone the Repository
```bash
git clone https://github.com/zwang2359/Nonunique-decompositions-of-mixed-states-and-deterministic-energy-transfers-simulation-code.git
cd Nonunique-decompositions-of-mixed-states-and-deterministic-energy-transfers-simulation-code
```

### 2. Set Up the Environment
It is recommended to use a virtual environment or an Anaconda environment. Install the required dependencies using `pip`:
```bash
pip install -r requirements.txt
```
